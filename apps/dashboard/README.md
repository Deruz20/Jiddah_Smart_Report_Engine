
  # Execute Action

  This is a code bundle for Execute Action. The original project is available at https://www.figma.com/design/1yvV5jQuBZaPG2IUK8uvBR/Execute-Action.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Environment

  Copy `apps/dashboard/.env.example` to `apps/dashboard/.env.local` if you need local env values.
  In development, the dashboard proxies `/api` and `/admin` to `http://localhost:3000`.
